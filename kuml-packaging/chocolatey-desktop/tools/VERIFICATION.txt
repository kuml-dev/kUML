VERIFICATION

Verification is intended to assist the Chocolatey moderators and community in
verifying that this package's contents are trustworthy.

This package downloads the official kUML Desktop installer published by the
kUML project on GitHub Releases:

  https://github.com/kuml-dev/kuml/releases

The downloaded file is:

  kuml-desktop-<version>.msi

The SHA-256 checksum embedded in tools\chocolateyInstall.ps1 is computed at
release time from the exact artifact attached to the GitHub Release of the
matching tag (v<version>), by the publish-chocolatey-desktop job in:

  https://github.com/kuml-dev/kuml/blob/master/.github/workflows/release.yml

To verify manually:

  1. Download the MSI from the GitHub Release for the version you installed.
  2. Compute its SHA-256:
        PowerShell:  Get-FileHash kuml-desktop-<version>.msi -Algorithm SHA256
  3. Compare it to the `checksum64` value in tools\chocolateyInstall.ps1.

The software is published by Irakli Betchvaia / kuml-dev under the Apache
License 2.0. The MSI is produced by jpackage (Compose Multiplatform's
nativeDistributions) and is unsigned in this release — Authenticode signing
is a tracked Phase-2 item alongside the project's macOS
notarization/signing work.
