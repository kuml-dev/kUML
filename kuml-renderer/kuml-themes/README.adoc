= kuml-themes
:toc:

image:../../docs/images/kuml-logo-simple.png[kUML,width=48,role=left] *kuml-themes*

Compose Multiplatform adapter for `kuml-themes-core`.
Converts framework-neutral `KumlTheme` data into Compose `MaterialTheme`-compatible
types via the `ThemeAdapter` extension functions (`.toCompose()`).

Used by the `kuml-kuiver` Compose renderer backend.
Direct SVG/PNG renderers (`kuml-io-svg`, `kuml-io-png`) use `kuml-themes-core` only.

== Key Types

[cols="1,3"]
|===
| Type | Description

| `ComposeKumlTheme`
| Compose-native theme wrapper. Passed to `KumlKuiverRenderer`.

| `ThemeAdapter.kt`
| `fun KumlTheme.toCompose(): ComposeKumlTheme` — bridges the two worlds.
|===

== Usage

[source,kotlin]
----
val composeTheme = PlainTheme().toCompose()
KumlKuiverRenderer.render(diagram, layoutResult, composeTheme)
----

== Installation

[source,kotlin]
----
dependencies {
    implementation("dev.kuml:kuml-themes:0.24.0")
}
----
