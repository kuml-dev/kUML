= kuml-llm-core
:toc:

image:../../docs/images/kuml-logo-simple.png[kUML,width=48,role=left] *kuml-llm-core*

LLM backend abstraction for kUML.

Defines the provider-agnostic contract used by `kuml-llm-bench` and any other
kUML feature that needs to call a large-language model. Pure Kotlin, no
third-party HTTP or vendor SDK dependencies — concrete backends live in
sibling modules.

Commit: `fc1a8ba`.

== Key Types

[cols="1,3"]
|===
| Type | Description

| `LlmBackend`
| Provider-agnostic backend interface: `suspend fun complete(messages: List<LlmMessage>): LlmResponse`.

| `LlmMessage`
| Role + content pair (`SYSTEM`, `USER`, `ASSISTANT`).

| `LlmResponse`
| Completion text + optional token-usage metadata.

| `LlmMockBackend`
| Deterministic mock backend (returns canned responses keyed by prompt prefix) for unit tests and offline benchmark runs.

| `LlmException`
| Unrecoverable backend error (HTTP failure, malformed response, rate-limit, …).
|===
