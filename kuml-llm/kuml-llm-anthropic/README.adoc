= kuml-llm-anthropic
:toc:

image:../../docs/images/kuml-logo-simple.png[kUML,width=48,role=left] *kuml-llm-anthropic*

Anthropic Claude API backend for `kuml-llm-core`.

Implements `LlmBackend` using the JDK 11+ `java.net.http.HttpClient` — no
extra dependencies (no Retrofit, no OkHttp, no Anthropic SDK). Keeps the
classpath small and GraalVM-friendly.

Commit: `fc1a8ba`.

== Configuration

[cols="1,3"]
|===
| Setting | Default

| Model
| `claude-3-5-haiku-20241022`

| API endpoint
| `https://api.anthropic.com/v1/messages`

| API key env var
| `ANTHROPIC_API_KEY`

| API version header
| `anthropic-version: 2023-06-01`
|===

== Usage

[source,kotlin]
----
val backend: LlmBackend = AnthropicBackend(
    apiKey  = System.getenv("ANTHROPIC_API_KEY"),
    modelId = "claude-3-5-haiku-20241022"
)

val response = backend.complete(
    listOf(
        LlmMessage(LlmRole.SYSTEM, "You are a kUML model generator."),
        LlmMessage(LlmRole.USER,   "Generate an Order class with id and status.")
    )
)
println(response.text)
----

== Dependencies

* `kuml-llm-core`
* JDK 11+ (`java.net.http.HttpClient`)
* `kotlinx-serialization-json` (request/response shape)
