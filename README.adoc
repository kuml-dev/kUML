= kUML
:toc:
:toc-placement: preamble
:icons: font

image::docs/images/kuml-banner.png[kUML Banner,link=https://github.com/kuml-dev/kUML]

*kUML* ist ein Modellierungswerkzeug, das UML 2.x, SysML 2 und C4 als
type-safe Kotlin-DSL ausdrückt — das erste UML-Werkzeug, das bewusst
für die LLM-Ära entworfen wurde.

image:https://img.shields.io/maven-central/v/io.kuml/kuml-core[Maven Central]
image:https://img.shields.io/github/license/kuml-dev/kUML[Apache 2.0]
image:https://img.shields.io/github/actions/workflow/status/kuml-dev/kUML/ci.yml[CI]

== Quick Start

[source,kotlin]
----
// hello.kuml.kts
diagram(name = "Hello kUML", type = DiagramType.CLASS) {
    // Phase 1: classOf, association, etc.
}
----

[source,bash]
----
kuml render hello.kuml.kts --format svg
----
