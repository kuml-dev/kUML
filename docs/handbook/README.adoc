= kUML Handbook
:icons: font

The Antora-based user handbook for kUML. Browse the source at
link:modules/[modules/], or build the site locally.

== Build the site

[source,bash]
----
npm install -g @antora/cli @antora/site-generator
antora generate docs/handbook/antora-playbook.yml --to-dir docs/handbook/build/site
open docs/handbook/build/site/index.html
----

(`open` is macOS. Use `xdg-open` on Linux, `start` on Windows.)

== Contributing

Pages are plain AsciiDoc. Cross-references use Antora's `xref:module:page.adoc[]` syntax:

* Within the same module: `xref:other.adoc[Label]`
* Across modules: `xref:tooling:cli.adoc[CLI]`

Avoid hard URLs to GitHub; use Antora attributes (defined in `antora.yml` -> `asciidoc.attributes`)
for the project URL and site URL:

[source,asciidoc]
----
See {project-url}[the repo] for the source.
----

== Status

This handbook covers kUML v{kuml-version}. It is in active development and ships with
every release. Sections may move, but URLs are stable — Antora's slugged URLs are
generated from page filenames, which are deliberately conservative.
