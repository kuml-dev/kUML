= kuml-codegen-api
:toc:

image:../../docs/images/kuml-logo-simple.png[kUML,width=48,role=left] *kuml-codegen-api*

Plugin API for kUML code generators.

Defines the stable contract that every code-generation plugin must implement.
Pure Kotlin, no third-party dependencies — safe to depend on from any kUML
module without pulling in target-language tooling.

Commit: `0df0891`.

== Key Types

[cols="1,3"]
|===
| Type | Description

| `KumlCodeGenerator`
| Plugin interface. Receives a fully evaluated `UmlModel` (plus optional plugin options) and returns a list of generated source files.

| `CodeGenerationException`
| Thrown by generators on unrecoverable errors. Carries the offending element's qualified name when known.

| `GeneratedFile`
| Output record: target relative path + UTF-8 source string.
|===

== Implementations

* link:../kuml-gen-kotlin[`kuml-gen-kotlin`] — built-in Kotlin generator (data classes, interfaces, enums).

== Roadmap
